## About The App
jQuery-sovellus näyttää Finnkinon elokuvatarjontaa ja tarjoaa lisäksi mahdollisuuden hakea elokuvia OMDb:stä. Projekti on toteutettu JavaScriptin sijaan jQueryllä.
